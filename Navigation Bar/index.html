Testing Navigation Bar!

Hi, my name is Edmund Frio and I am a sophomore at Kapi'olani Community College and I am majoring in Information Technology. I was into computers for a while and decided that since the field of technology has been growing immensely, I'd give it a chance and all types of careers it offers. Anything that has to do with programming has intrigued me so I stuck to this path ever since. What you see here is my first programming related project that uses HTML, CSS and JavaScript. I'm still a beginner when it comes to effectively writing code for these three languages, but it has been a blast studying and putting my skills to use after sticking with Python for about a year. Being a full-time student is pretty hard especially when all my classes are all online so chasing this opportunity of testing myself alongside other people who want to do the same is going to be extremely beneficial down the road. Getting my foot in the door is going to be a big milestone for me and I'm glad to being do it at a time like this.
